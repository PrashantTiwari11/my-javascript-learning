Window Object -: 
It is object's browser (not javascript) and is automatically created by browser.

What is DOM (Dociment Object Model)?
When a webpage is loaded , the browser creates a DOM of the page.
Tree Structure -:
window --> document --> html
body <--  html --> head
head --> meta , title , link
body --> div , script

DOM Manipulation :
1) Selecting With ID
document.getElementId("myId");

2) Selecting With ClassName
document.getElementsByClassName("myClassName");
Note : It returns HTML collections.

3) Selecting With Tag
document.getElementsByTagName("p");

<--- QUERY SELECTOR -->
document.querySelector("id" | "className" | "tagName"); //returns first elements 
~~~~
document.querySelectorAll("id" | "className" | "tagName");//returns all elements

Attributes -:
1) getAttribute(attr) // to get the attribute value
2) setAttribute(attr , val) // to set the attribute value

Style -:
: node.style

let el = document.createElement("div"); // first create that elements
~~~~~~
Insert elements -: 
: node.append(el) // adds at the end of node (inside)
: node.prepend(el) // adds at the start of node (inside)
: node.after(el) // adds after the node (outside)
: node.before(el) // adds before the node (outside)

Delete Element -:
: node.remove(el) // remove the node
